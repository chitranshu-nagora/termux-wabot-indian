## Clone this project

```bash
> https://github.com/chitranshu-nagora/termux-wabot-indian
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd termux-wabot
> bash install.sh
```

### Usage
```bash
> npm start
```

### Settings
```json
{
	"prefix": "#",
	"ownerNumber": "62858xxxxx",
	"apiKey": "Your ApiKey here"
}
```
